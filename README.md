# EFT Kappa Collector Checklist

Unofficial fan-made checklist for Escape from Tarkov Kappa Collector items.

## Use online

Open the GitHub Pages version.

## Use as a local HTML file

Download `eft-kappa-collector-checklist.html` from the latest release and open it in your browser.

## Features

- Collector item checklist
- Local progress auto-save
- Search
- Missing / Completed filters
- Export / Import backup
- Item icons when online

## Notes

- Progress is stored only in your browser.
- Use Export before clearing browser data or moving to another device.
- Item icons are loaded online when available.
- This is an unofficial fan-made tool.
